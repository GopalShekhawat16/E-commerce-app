## Run Locally

Clone the project

```bash
  git clone https://github.com/GopalShekhawat16/E-commerce-app
```

Go to the project directory

```bash
  cd E-commerce-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
